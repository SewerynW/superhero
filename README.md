# superhero

You can check it on: https://angry-meitner-98fabf.netlify.app/

## Build Setup

```bash

# download project
$ git clone https://github.com/SewerynW/superhero.git

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate

# test
$ npm run test
```
